Usage
-----
+-------+--------------------------------------+
| Key   | Action                               |
+=======+======================================+
| Space | Start/Stop                           |
+-------+--------------------------------------+
| r     | Reset                                |
+-------+--------------------------------------+
| t     | Swap between fullscreen and windowed |
+-------+--------------------------------------+
| Esc   | Exit                                 |
+-------+--------------------------------------+
