stopwatch
=========
pygame-based stopwatch with Python 3 support.

.. image:: https://raw.github.com/myint/stopwatch/master/screenshot.png
   :alt: screenshot
   :align: center

Usage
-----
+--------+--------------------------------------+
| Key    | Action                               |
+========+======================================+
| Space  | Start or stop                        |
+--------+--------------------------------------+
| r      | Reset                                |
+--------+--------------------------------------+
| f      | Swap between fullscreen and windowed |
+--------+--------------------------------------+
