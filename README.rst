Usage
-----
+--------+--------------------------------------+
| Key    | Action                               |
+========+======================================+
| Space  | Start/Stop                           |
+--------+--------------------------------------+
| r      | Reset                                |
+--------+--------------------------------------+
| f      | Swap between fullscreen and windowed |
+--------+--------------------------------------+
| Escape | Exit                                 |
+--------+--------------------------------------+
